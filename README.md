# 🕹️ Python Games & Mini-Projects

A collection of interactive games and logic-based projects developed while mastering Python fundamentals.

## 🚀 Featured Games
*   **Ping Pong:** A classic 2-player arcade game. (Requires `pygame`)
*   **Hangman:** A word-guessing game with custom logic.
*   **Number Guesser:** A game focused on binary search logic and user input handling.

## 🛠️ Concepts Applied
- **Logic Flow:** Nested loops, conditionals, and error handling.
- **Data Structures:** Lists and Dictionaries for word banks and scores.
- **Algorithms:** Implementation of randomization and game loops.

## ⚙️ How to Run
1. Clone the repository.
2. Install dependencies (if any): `pip install -r requirements.txt`
3. Run any script: `python game_name.py`
