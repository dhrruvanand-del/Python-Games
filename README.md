# 🎮 Python-Games

A collection of **classic games** built with **Python** and **Pygame**. Perfect for learning game development, experimenting with logic, or just having fun!

---

## 📌 **Games Included**


| Game                | Description                                       | Run Command                 | Screenshot      |
| ------------------- | ------------------------------------------------- | --------------------------- | --------------- |
| **Hangman**         | Guess the word before the stickman dies!          | `python hangman.py`         | Hangman         |
| **Ping Pong**       | 2-player paddle game with score tracking          | `python pingpong.py`        | Ping Pong       |
| **Number Guessing** | Guess the random number generated by the computer | `python number_guessing.py` | Number Guessing |


---

## 📦 **Requirements**

Install the required packages using:

```bash
pip install -r requirements.txt
```

or manually:

```bash
pip install pygame
```

---

## 🚀 **How to Run**

1. Clone the repository:
  ```bash
   git clone https://github.com/your-username/Python-Games.git
   cd Python-Games
  ```
2. Install dependencies (see above).
3. Run any game:
  ```bash
   python hangman.py
  ```
